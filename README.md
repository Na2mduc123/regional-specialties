# Đây là hướng dẫn sử dụng

- Dự án gồm 3 phần:

## Frontend:

- Ngôn ngữ sử dụng: Javascript nâng cấp (TypeScript)
- Thư viện xây dựng: ReactJs
- Framework: Tailwind CSS
- Công cụ build: Vite

### Gồm các thư mục và file quan trọng sau:

- index.html
- src

### Chạy dự án (Với máy client)

- yarn
- yarn dev

## Backend:

- NodeJs

## Database
